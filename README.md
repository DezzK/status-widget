# Виджет информации (Status Widget)

Плавающий статус-виджет для **Android-головных устройств в автомобилях**.
Показывает поверх любого приложения время, дату, текущий трек, состояние
Wi-Fi, GPS и Bluetooth. Собирается «из кирпичиков» — пользователь сам решает,
что показывать и в каком порядке.

Приложение целенаправленно сделано под ГУ (Geely Monjaro, китайские
универсальные «головы» и т. п.) с Android 9–12. Не публикуется в Google Play
и устанавливается только сторонней установкой (sideload).

## Возможности

### Содержимое виджета — конструктор «из кирпичиков»

Виджет собирается из независимых элементов (bricks), которые можно добавлять,
удалять и перетаскивать в произвольном порядке:

- **Время** — текущее время, формат подстраивается под системные настройки.
- **Дата** — день месяца, день недели; возможна однострочная или двустрочная
  раскладка, полные или сокращённые названия, порядок «число → день недели»
  или наоборот.
- **Воспроизведение** — текущий трек: исполнитель и название с прокруткой
  (marquee) для длинных строк, плюс опциональная подпись с именем приложения-
  источника. Поддерживается любой плеер, выставляющий стандартную
  `MediaSession`. Плейсхолдеры вроде «Unknown artist» / «Неизвестный
  исполнитель» автоматически игнорируются.
- **Wi-Fi** — иконка состояния сети.
- **GPS** — иконка статуса геолокации, опциональный значок числа спутников
  (требует GNSS Share).
- **Bluetooth** — три состояния (выкл / нет подключённых устройств /
  подключено), опциональный значок числа подключённых устройств.

### Два режима виджета

- **Плавающий.** Виджет можно перемещать по экрану перетаскиванием.
  Поддерживается «привязка к правому краю»: при изменении ширины
  содержимого правая граница остаётся на месте.
- **Статусная строка.** Виджет растягивается на всю ширину экрана и
  закрепляется сверху. Каждому кирпичику отдельно назначается выравнивание
  (слева / по центру / справа).

### Внешний вид

- **Тема виджета.** Системная / светлая / тёмная / инверсная — независимо от
  темы остального устройства.
- **Дизайн иконок.** Классический / силуэт / полоски.
- **Стиль иконок.** Монохромные или цветные.
- **Фон.** Регулируемая прозрачность и скругление углов.
- **Отступы.** Настраиваются по всем четырём сторонам.

### Тонкая настройка каждого кирпичика

Для каждого элемента отдельно регулируются:

- Размер шрифта или иконки.
- Семейство шрифта (Roboto Condensed, Sans Serif, Serif, Monospace, и др.)
  плюс независимые жирное начертание и курсив.
- Прозрачность и толщина обводки текста / иконок (в пикселях).
- Отступы слева и справа.
- Сдвиг по вертикали (мелкая подгонка для выравнивания между разнотипными
  элементами).
- Максимальная ширина (только для медиа — чтобы длинный трек не растягивал
  весь виджет).
- Выравнивание в статусной строке.

### Скрытие в отдельных приложениях

У каждого кирпичика свой список приложений, поверх которых он не показывается.
Список можно «унаследовать» от другого кирпичика, чтобы не дублировать выбор
для всех элементов сразу.

### Пресеты конфигурации

- **Встроенные пресеты** — несколько готовых вариантов внешнего вида и набора
  кирпичиков.
- **Пользовательские пресеты** — можно сохранить текущее состояние под
  именем, экспортировать в файл и поделиться с другими пользователями;
  обратный импорт чужого пресета — в один клик.

### Прочее

- **Экспорт и импорт настроек** — целиком, как JSON-файл.
- **Сброс настроек** — глобальный (с подтверждением) и отдельно для каждого
  кирпичика.
- **Старт виджета до разблокировки.** Виджет автоматически поднимается
  после загрузки головного устройства, даже если экран ещё заблокирован.
- **Отчёт о крашах.** Если приложение аварийно завершилось, при следующем
  запуске покажется диалог со стек-трейсом и кнопками «Копировать» /
  «Отправить» / «Отклонить».
- **Локализация** — русский и английский.

## Системные требования

- Android 9, 10, 11 или 12 на головном устройстве.
- Около 5 МБ свободного места.
- Подключение к интернету **не** требуется для работы виджета (нужно только
  для отображения полной страницы «О приложении»).

## Разрешения

Для полноценной работы виджет запрашивает:

| Разрешение | Зачем нужно |
|---|---|
| Поверх окон (`SYSTEM_ALERT_WINDOW`) | Сам факт отрисовки виджета поверх других приложений. |
| Геолокация и геолокация в фоне | Чтобы фоновая служба продолжала работать после смены приложения. |
| Доступ к статистике использования (`PACKAGE_USAGE_STATS`) | Чтобы виджет понимал, какое приложение сейчас на переднем плане, и скрывался в указанных приложениях. |
| Доступ к уведомлениям (`Notification Listener`) | Нужен только для отображения «Сейчас играет», читает активную медиа-сессию. |

### Автоматическая выдача прав (для урезанных ГУ)

На некоторых головных устройствах (например, Geely Monjaro) системное
приложение «Настройки» лишено экранов, через которые пользователь обычно
выдаёт эти разрешения вручную. Поэтому при запуске виджет:

1. Сканирует локальные сетевые интерфейсы в поисках открытого ADB
   (порты 5555 и 7777) или Telnet (порт 23).
2. Если канал найден — подключается к нему и выдаёт себе нужные разрешения
   через `appops`, `pm grant` и `cmd notification` от имени `shell`.
3. Сообщает результат тостом «Разрешения получены автоматически: …».

На обычном смартфоне или планшете этот шаг тихо пропускается, и разрешения
запрашиваются стандартным образом — через системные диалоги.

## Установка

1. Скачать APK из раздела
   [Releases](https://github.com/DezzK/status-widget/releases).
2. Скопировать на головное устройство (по USB, через файловый менеджер,
   через ADB — как удобнее).
3. Разрешить установку из неизвестных источников и установить APK.
4. Открыть приложение, разрешить запрошенные права. На ГУ с открытым ADB
   разрешения выдадутся автоматически; на остальных устройствах появятся
   стандартные системные диалоги.
5. Включить виджет переключателем в верхней части экрана настроек.
